Objective

See which nutritional intake affects recovery.

Methodology

The methodology appraoch of this project focuses on cleaning, then imputating the data. From there feature selection was conducted and then skewness and outlier tests were conducted. From there models were developed and the two champion models were tree based regression models. We then tuned the parameters to get the most optimal results.

Recomendation
